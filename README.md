# Calculadora de Costes y Rentabilidad (España → Venezuela)

Calculadora para importadores: compra en EUR, venta en Venezuela con doble tasa (BCV / Binance).

- Convierte el coste EUR → USD con la tasa del día (BCE / Frankfurter).
- Carga automática de tasas BCV y Binance/Paralelo (DolarAPI Venezuela), editables a mano.
- Barra de ganancia deseada (10–100%).
- Calcula el precio en Bs usando la tasa Binance, para que al recomprar dólares se conserve la ganancia real.
- Panel lateral con productos guardados (localStorage).

## Uso

Abrir `index.html` en el navegador, o hostear como sitio estático (Vercel, GitHub Pages, etc.).
